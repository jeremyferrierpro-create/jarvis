================================================================
  FICHIER credentials.json — INSTRUCTIONS
================================================================

Ce fichier (credentials.json) est REQUIS pour que JARVIS puisse
accéder à vos services Google (Gmail, Drive, Docs, Sheets,
Calendar).

COMMENT L'OBTENIR :
1. Allez sur : https://console.cloud.google.com/
2. Créez un projet ou sélectionnez le vôtre.
3. Activez les APIs suivantes :
   - Gmail API
   - Google Drive API
   - Google Docs API
   - Google Sheets API
   - Google Calendar API
4. Allez dans "Identifiants" → "Créer des identifiants"
   → "ID client OAuth 2.0" → Application Bureau
5. Téléchargez le fichier JSON.
6. Renommez-le exactement : credentials.json
7. Placez-le dans ce dossier (à côté de main2.py).

NOTE : Sans ce fichier, JARVIS fonctionnera quand même,
       mais les fonctions Gmail, Drive, Docs, Sheets et
       Calendar seront désactivées.
